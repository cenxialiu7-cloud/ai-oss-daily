# Claude Skill 策展包 + 本機 Skill 管理器（吃 skill 生態爆發紅利）

OSS Radar 每天幫你挑出爆紅的 Claude/OpenClaw skill，你品質篩選+繁中化打包成主題包賣，並配一個本機管理 App。

**怎麼變現**：①主題 skill pack 訂閱（行銷/研究/資安/設計）②本機 skill 管理/執行 Mac App 買斷＋Ko-fi ③企業客製 skill 包。OSS Radar 本身就是你的免費選品雷達。

**運作方式**：用 OSS Radar 每日榜鎖定單日爆紅 skill → 人工品質篩選＋繁中化＋去重 → 打包成主題 pack → 做一個本機 Mac App 統一安裝/管理/執行這些 skill（參考 open-design 的本機桌面做法）。

**難度**：medium · **投入估計**：3–4 週；策展靠 OSS Radar 半自動，主要工在打包與 App 殼。
