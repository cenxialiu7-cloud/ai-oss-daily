AI 短影音工廠（批量產內容 → 流量變現）

一句話腳本 → 自動產出帶配音、配樂、封面的成品短影音，規模化經營流量帳號。

Difficulty: medium 2–3 週做出能跑的 pipeline；難點在剪接串接與發佈自動化，模型都現成。

Monetization

①自營 TikTok/Reels/Shorts 帳號衝流量，影片描述掛 VPN affiliate（NordVPN/Surfshark）②代製接案 ③模板/prompt 包販售。符合『次收入用 Adsterra』，廣告只放網頁端。

How it works

選題 agent 產腳本 → Sulphur-2 生影片片段 → supertonic 配旁白 → stable-audio-3 配樂 → Lens 生封面 → 自動剪接上字幕 → 排程發佈。lanshu kit 提供現成 prompt 庫加速。
