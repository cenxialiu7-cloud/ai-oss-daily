# ARSreg46/Ai-Image-Generator

AI Image Generator is a powerful desktop application for creating stunning AI-generated artwork from text prompts. It integrates with Stable Diffusion XL, Flux, DALL-E 3, and Midjourney APIs for both local GPU-accelerated and cloud-based image generation.
