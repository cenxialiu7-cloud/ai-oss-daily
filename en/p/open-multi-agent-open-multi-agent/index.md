# open-multi-agent/open-multi-agent

TypeScript multi-agent orchestration framework. Describe a goal, a coordinator decomposes it into a task DAG that runs on any LLM: Claude, GPT, Gemini, DeepSeek, or local models. Model-agnostic and MCP-ready.
